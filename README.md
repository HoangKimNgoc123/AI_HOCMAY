### Hướng dẫn chạy chương trình
1. Hướng dẫn chạy huấn luyện mô hình
   - Bước 1: Clone source code về máy và mở folder Train_Model
   - Bước 2: Mở và chạy file AI_Du_doan_gia_nha.ipynb trên Google Colab hoặc Jupyter Notebook, đảm bảo có file train.csv đi kèm
3. Hướng dẫn chạy Web
   - Bước 1: Clone source code về máy và mở folder Web_App
   - Bước 2: Cài đặt thư viện
    'pip install pandas numpy scikit-learn xgboost streamlit joblib'
  - Bước 3: Chạy ứng dụng Web
    Tại thư mục gốc của dự án, khởi động server Streamlit bằng lệnh:
      'streamlit run app.py'
    - Bước 4: Trình duyệt sẽ tự động mở tại địa chỉ
      'http://localhost:8501'
